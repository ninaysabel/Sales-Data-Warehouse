# Sales-Data-Warehouse

Integrated Sales Data Warehouse is a cloud-based data engineering project that integrates customer, sales, and product data from the FudgeMart and FudgeFlix business systems into a centralized analytics platform. Using Snowflake, dbt, SQL, and Power BI, this project demonstrates how modern data warehousing and business intelligence practices transform fragmented data into actionable insights that support reporting, customer analysis, and strategic decision-making.
